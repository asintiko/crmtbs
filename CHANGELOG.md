# Список изменений

## Кликабельный тип "Бронь" в журнале операций

**Что добавлено:**
- Тип операции "Бронь" в таблице журнала теперь кликабельный
- При клике открывается модальное окно с детальной информацией о бронировании

**Как использовать:**
1. Откройте страницу "Журнал"
2. Найдите операцию с типом "Бронь" (желтый бейдж)
3. Кликните на бейдж "Бронь"
4. В модальном окне увидите: клиента, контакт, количество, срок действия, комментарий, код бронирования
5. Можно скопировать ссылку на бронь или перейти на страницу бронирований

## Просмотр остатков по моделям раций

**Что добавлено:**
- Новая страница "По моделям" в меню (между "Склад" и "Журнал")
- Группировка товаров по полю "Модель" с суммированием остатков
- Возможность раскрыть детали по каждому товару в модели

**Как использовать:**
1. Откройте раздел "По моделям" в меню
2. Увидите таблицу со всеми моделями и суммарными остатками
3. Кликните на строку модели, чтобы раскрыть список товаров этой модели
4. Для каждого товара отображаются индивидуальные остатки
5. Используйте поиск для быстрого нахождения модели или товара

**Важно:** Товары без указанной модели не отображаются на этой странице.

## Системные напоминания (уведомления на ПК)

**Что добавлено:**
- Кнопка "Добавить" на странице Настроек для создания напоминаний
- Модальное окно с быстрыми шаблонами времени
- Автоматическое создание напоминаний при определенных операциях
- Системные уведомления ПК в назначенное время

**Как использовать:**
1. Откройте "Настройки"
2. Найдите раздел "Напоминания (системные уведомления)"
3. Нажмите кнопку "Добавить"
4. Заполните заголовок, выберите время (или используйте быстрые шаблоны: "Сегодня вечером", "Завтра утром", "Через час", "Через день")
5. При необходимости добавьте комментарий
6. Нажмите "Создать напоминание"

**Автоматические напоминания:**
- При создании брони с датой окончания - автоматически создается напоминание за день до окончания
- При отгрузке в долг - автоматически создается напоминание через 7 дней о проверке оплаты

Напоминания проверяются каждую минуту и показываются через системные уведомления операционной системы.

## Примечания к товарам на складе

**Что добавлено:**
- Улучшенное отображение примечаний в таблице склада
- Модальное окно для просмотра и редактирования примечаний
- Возможность быстро добавить примечание, если его нет

**Как использовать:**
1. На странице "Склад" найдите товар
2. В колонке "Примечания" кликните на примечание (или кнопку "Добавить" если его нет)
3. В модальном окне увидите полный текст примечания
4. Нажмите "Редактировать" для изменения или добавления текста
5. Сохраните изменения

## Групповая операция (комплект товаров)

**Что добавлено:**
- Форма для создания операций с несколькими товарами одновременно
- Автоматическое связывание операций в один комплект
- Визуальная группировка операций комплекта в журнале

**Как использовать:**
1. На странице "Журнал" или "Склад" нажмите кнопку "Комплект товаров"
2. Выберите тип операции (Продажа, Отгрузка в долг, Бронь и т.д.)
3. Укажите название комплекта (например, "Отгрузка для ООО Компания")
4. Заполните клиента, контакт и другие общие параметры
5. В поле поиска найдите и добавьте товары (можно добавить несколько)
6. Для каждого товара укажите количество
7. Нажмите "Создать операции"
8. Все операции будут созданы одновременно и связаны в один комплект

**В журнале:**
- Операции одного комплекта отображаются вместе
- Над группой операций показывается заголовок с названием комплекта
- Операции комплекта выделены фоном
- В колонке "Комплект" отображается бейдж с названием

Это удобно для отгрузки нескольких позиций одному клиенту - все операции создаются одной командой.
